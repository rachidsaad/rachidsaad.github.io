about

introduction:

Slave Tasks takes the latest posts from r/slavelabour using the reddit api, then render them with style, that sums it up (stop reading).

Made this just so I have more control over the layout, easy browsing of latest tasks, with colors that will not hurt my eyes, a friendly (I think) user interface (UI / this is nice for SEO).

Started this on my own selfish reason, but while making it, I was like, I think there are other slaves out there that are like me struggling with the general layout of r/slavelabour (custom css | mainly old reddit , just filter by new and you will see... Links color with a contrast of more than 2.55 on a blurry background...) so I said why not add a few features then share it with others... so here we are.

How do I use it:

  • Browse the tasks.
  • Mark if interested (the title of the task lives in your local storage, so even if you close the page, it will still be marked when you come back).
  • Select task.
  • Place a bid.
  • PM the user.
  • Do the usual.
